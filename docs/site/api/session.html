    session¶

    Sessions, manifests, and artifact-oriented persistence helpers.

    Entrypoints¶

    • session.Session

    Package Export¶

    musiq.session ¶

    Session storage public API.

    The session package provides revision-oriented storage helpers for workflow artifacts. The primary public entrypoint is Session.

    Session ¶

    Versioned artifact session with manifest-based indexing.

    Example 
    from musiq.session.session import Session

s = Session.open("runs/session_a")
rev = s.commit("circuit", {"schema_version": "1.0", "qasm": "OPENQASM 3;"})
print(s.get(rev))

    commit(kind, payload, *, inputs=None, dependencies=None, tags=None) ¶

    Store an artifact revision and append metadata to session manifest.
