    qec¶

    Package Export¶

    musiq.qec ¶

    QEC public API.

    This package collects the main prior builders, decoders, and helper functions used by the workflow QEC stages. It is the primary API surface for:

    • selecting a decoder implementation
    • building priors from syndrome data
    • summarizing logical error outputs
