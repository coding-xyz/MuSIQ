    circuit¶

    OpenQASM import, export, and circuit normalization.

    Entrypoints¶

    • circuit.CircuitAdapter

    Package Export¶

    musiq.circuit ¶

    Circuit public API.

    The circuit package currently exposes helpers for importing, exporting, and normalizing OpenQASM-based circuit descriptions. The primary public entrypoint is CircuitAdapter.

    CircuitAdapter ¶

    Adapter between OpenQASM/Qiskit and CircuitIR.
