    ModelSpec dataclass ¶

    Engine-neutral simulation model specification.

    ModelSpec is the structured boundary between backend lowering and numerical engines. It describes the circuit context, solver request, time grid, frame, physical system, Hamiltonian, noise, readout, and analysis request without depending on a backend-private runtime representation.

    Attributes:

    Name Type Description
    circuit 'CircuitSpec | None'

    The circuit specification associated with the simulation.
    solver 'SolverSpec | str'

    Solver specification (e.g., SE, ME, SME).
    time 'TimeSpec'

    Time grid specification (dt, t_end).
    frame 'FrameSpec'

    Reference frame and RWA settings.
    system 'SystemSpec'

    Physical system description (qubits, resonators).
    hamiltonian 'HamiltonianSpec'

    System Hamiltonian including controls and couplings.
    noise 'NoiseSpec'

    Noise model and dissipation channels.
    readout 'ReadoutSpec | None'

    Readout protocol and chain specification.
    analysis_request 'AnalysisRequestSpec | None'

    Requested post-processing analysis.
    study 'StudySpec | None'

    Study context if this is part of a parameter sweep.
    metadata dict[str, Any]

    Non-primary technical annotations and debug notes.
